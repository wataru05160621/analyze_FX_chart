# FX分析システム API費用削減計画

## 現状の問題点

### 月額API費用（現在）
- **Claude 3.5 Sonnet**: $104.55/月（約15,682円）
- **GPT-4o**: $86.00/月（約12,900円）

### 高額になっている原因
1. **PDFファイル**: 80,000文字（約200,000トークン）
2. **1日5回実行**: 月間3,300万トークン
3. **画像サイズ**: 各200KB × 2枚

## 削減策

### 1. 即効性のある対策（実装済み）

#### A. PDFの最適化
- 重要セクションのみ抽出：80,000文字 → 10,000文字
- トークン削減：200,000 → 25,000トークン（87.5%削減）

#### B. プロンプトの簡潔化
- 分析指示：6,000文字 → 1,000文字
- トークン削減：15,000 → 2,500トークン（83%削減）

#### C. 画像の圧縮
- リサイズ：最大1280x720ピクセル
- ファイルサイズ：200KB → 100KB以下
- トークン削減：2,338 → 1,200トークン（49%削減）

#### D. 出力の制限
- max_tokens：4,000 → 2,000
- 実際の出力：約1,000文字の簡潔な分析

### 2. 削減後の費用予測

#### トークン使用量（1回あたり）
- **削減前**: 220,000トークン
- **削減後**: 30,000トークン（86%削減）

#### 月間費用
| API | 削減前 | 削減後 | 削減額 |
|-----|--------|--------|--------|
| Claude 3.5 | $104.55 | **$14.25** | $90.30（86%削減） |
| GPT-4o | $86.00 | **$11.70** | $74.30（86%削減） |

### 3. 追加の削減オプション

#### A. 実行頻度の最適化
- 週末は実行しない（月20日運用）
- 費用をさらに33%削減

#### B. 分析対象の絞り込み
- 8時：USD/JPYのみ詳細分析＋ブログ
- 15時、21時：簡易分析モード
- BTC、XAUは週1回の分析

#### C. 代替モデルの検討
- Claude 3 Haiku（高速・低価格）
  - 入力：$0.25/100万トークン
  - 出力：$1.25/100万トークン
  - 月額予測：約$2-3
