# FXチャート分析システム向け定額制AIサービス比較 (2025年版)

## 現在の要件
- 画像分析機能（チャート画像2枚/回）
- 長文テキスト処理（PDF 80,000文字）
- 1日5回実行（月150回）
- 高品質な金融分析が必要

## 1. Claude Pro (Anthropic)

### 料金
- **月額**: $20/月（年払いは$17/月相当）
- **API料金**: 
  - Claude 3.5 Sonnet: $3/M入力トークン、$15/M出力トークン
  - Claude 4 Opus: $15/M入力トークン、$75/M出力トークン

### 機能・制限
- **利用制限**: 5時間あたり約45メッセージ（長さや負荷により変動）
- **画像分析**: ✅ 対応（画像、PDFの分析・要約が可能）
- **API利用**: ✅ 可能（別料金）
- **商用利用**: ✅ 可能
- **自動実行**: ✅ API経由で可能

### メリット・デメリット
- ✅ 高品質な分析（特に長文理解）
- ✅ プロジェクト機能で知識管理が可能
- ❌ Pro版はWebインターフェースのみ（APIは別料金）
- ❌ 5時間あたりの制限あり

### コスト試算（月150回実行の場合）
- APIのみ使用: 約$9.18/月（Claude 3.5 Sonnet使用時）
- Pro + API併用は推奨されない（Proは手動利用向け）

## 推奨構成

### FXチャート分析システムの要件に対して：

1. **開発・テスト段階**: 
   - Google Gemini API（無料枠）+ Claude API（従量課金）
   - コスト: 約$10〜20/月

2. **本番運用（コスト重視）**:
   - Claude 3.5 Sonnet API のみ
   - コスト: 約$9.18/月
   - 理由: 高品質な分析と低コスト

3. **本番運用（品質重視）**:
   - Claude 4 Opus API + バックアップとしてGemini API
   - コスト: 約$50〜100/月
   - 理由: 最高品質の分析と冗長性

4. **エンタープライズ向け**:
   - Azure OpenAI Service（バッチAPI使用）
   - コスト: 使用量に応じて変動（50%割引適用）
   - 理由: セキュリティとSLA保証

## 注意事項

1. **API vs サブスクリプション**: 
   - 自動実行にはAPI契約が必須
   - サブスクリプションはWebインターフェース利用のみ

2. **データプライバシー**:
   - 無料版・個人版ではデータがトレーニングに使用される可能性
   - エンタープライズ版は除外可能

3. **地域制限**:
   - 一部サービスは日本での利用に制限がある場合あり
   - 事前に利用可能地域を確認必要

4. **バッチ処理**:
   - リアルタイム不要なら50%割引のバッチAPIを活用
   - 24時間以内の処理で十分な場合に有効
