# 定額制を活用したFX分析システム代替案

## 現状の課題
- 従量課金で月額15,000円以上
- ボリューム（PDF 80,000文字）を削減したくない
- 自動実行が必要（API必須）

## 解決策1: Google Gemini無料枠の活用

### Gemini 1.5 Flash（無料枠）
- **料金**: 無料
- **制限**: 60クエリ/分、100万トークン/日
- **画像分析**: 対応
- **長文処理**: 最大100万トークン

### 実装方法
```python
# 1日5回実行 = 月150回
# 全て無料枠内で処理可能
```

### メリット
- 完全無料
- 制限内であれば80,000文字のPDFも処理可能
- 画像分析も高品質

### デメリット
- 将来的に無料枠が縮小される可能性
- 商用利用の制限に注意

## 解決策2: ハイブリッド型（推奨）

### 構成
1. **メイン**: Gemini無料枠（XAU/USD, BTC/USD）
2. **高優先度**: Claude 3 Haiku API（USD/JPYのみ）
3. **バックアップ**: OpenAI GPT-3.5 Turbo

### 月額コスト試算
```
Gemini: $0（無料枠）
Claude Haiku: 約$3（USD/JPYのみ）
合計: 約$3（450円）
```
