# FXチャート自動分析システム要件定義書

## 1. システム概要
USD/JPYの5分足チャートを定期的に取得し、Bob Volmanのスキャルピングメソッドに基づくAI分析を行い、その結果をNotionに記録する自動化システム

## 2. 機能要件

### 2.1 チャート取得機能
- **対象通貨ペア**: USD/JPY
- **時間足**: 5分足、1時間足
- **取得方法**: Playwright MCPを使用してスクリーンショット取得
- **取得タイミング**: 定時実行（時間は設定可能）

### 2.2 AI分析機能
- **利用サービス**: Claude 3.5 Sonnet API
- **分析内容**: 
  - Volmanメソッドによる6つのセットアップ（A-F）識別
  - 20/10ブラケットオーダーの設定
  - 25EMAを使用したトレンドバイアス分析
  - スキップルールの適用（ATR、7pips、スプレッド、2pips等）
- **プロンプトテンプレート**: Volmanメソッド準拠
- **分析モード**:
  - 通常分析: 実践的トレードプラン
  - ブログ分析: 教育的解説（投資助言なし）

### 2.3 結果保存機能
- **保存先**: Notion（将来的にブログ対応）
- **保存形式**:
  - チャート画像
  - AI分析結果テキスト
  - 実行日時
  - メタデータ

### 2.4 スケジューリング機能
- **実行頻度**: 設定可能（例：毎日9:00、15:00、21:00）
- **エラー時の再試行**: 自動リトライ機能

## 3. 非機能要件

- ログ記録機能

### 3.2 拡張性

## 4. システム構成

### 4.1 技術スタック
- **言語**: Python 3.x
- **スクレイピング**: Playwright
- **API連携**:
