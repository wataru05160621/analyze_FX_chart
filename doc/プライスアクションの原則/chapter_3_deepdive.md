## 3.0 概要

Chapter 3 は、プライスアクション“7原則”で得た概念を **リアルタイムの5分足チャート** 上で具体的にどう読み取り、どこで仕掛け・どこで逃げるかを解説する“橋渡し”パートです。主役は 25EMA と **Build‑Up → Break → Follow‑Through**（BBF）の 3 段階モデル。さらに、W ボトム / M トップ、フラッグ & ペナントなど主要フォーメーションを加え、“流れ” と “勢力交代点” を多角的に判断するスキルを身につけます。

## 3.1 25EMA — ダイナミック・バリアと環境認識

| 要素                     | 説明                                                                               |
| ---------------------- | -------------------------------------------------------------------------------- |
| **Concept**            | 25本指数移動平均を “動く支持・抵抗帯” として利用。価格が EMA の上下どちらに位置するかで優先すべき方向を判定し、EMA の傾きでトレンド強度を測る。  |
| **Chart Clues**        | *EMA をまたぐキャンドルが縮小 → 次足で実体が片側に張り付く* → 勢力が固まりつつあるサイン。傾きが±30°を超えると “順張り優位” フェーズに移行。 |
| **Trade Implications** | EMA 上側でのプルバックはロング再開の好機、下側ならショート再開。EMA 内包のスクイーズはブレイク待ち伏せ。                         |
| **Pitfalls**           | レンジ相場で EMA がヨコヨコの場合はダマシ多発 → 単独判断は禁物。                                             |

## 3.2 Build‑Up → Break → Follow‑Through（BBF）モデル

| ステージ               | 特徴                         | 典型キャンドル           | トレード戦略                      |
| ------------------ | -------------------------- | ----------------- | --------------------------- |
| **Build‑Up**       | 足10〜30本の狭いレンジ、出来高漸減、EMA 圧縮 | 小実体×連続            | “逆指値” を枠外1〜2 pipsに設置し待機     |
| **Break**          | 枠幅≥20 %を実体で抜く長いキャンドル       | エンゴルフィングバー／マラソンバー | 即成行 or 執行中の逆指値がヒット          |
| **Follow‑Through** | Break の実体幅×1〜2倍の伸び         | フロウキャンドル          | 利食い 20 pips 基準／伸びなければ裁量クローズ |

> ***Key Insight*** — **Build‑Up 内にさらに小さな Build‑Up** があるほど Break 後の Follow‑Through は鋭い。

## 3.3 W ボトム & M トップ — 勢力交代パターン

### W ボトム（ダブルボトム）

- **Concept** : 2 度目の安値試しが失敗 → ベア勢力の息切れを示す。2 度目のボトムで売りポジ閉鎖 + 買い新規参入の **ダブルの圧力** が発生。
- **Chart Clues** : 2 番底でヒゲ長 / 実体小 + 25EMA 上抜け。ネックライン幅≧10 pips。
- **Entry** : ネックラインブレイクでロング。ストップは 2 番底のヒゲ下。
- **TP** : ネックライン幅×2 or ラウンドナンバー。

### M トップ（ダブルトップ）

- **Concept** : 2 度目の高値試し失敗でブル勢力後退。
- **Chart Clues** : 天井で実体細り→ヒゲ伸長。25EMA 下抜け確定がトリガ。
- **Entry** : ネックライン割れでショート。ストップは 2 番天井ヒゲ上。

> **Pitfalls** — ネックライン付近がキリ番の場合、ブレイク幅が小さくダマシ率↑。

## 3.4 フラッグ & ペナント — トレンド継続パターン

| パターン       | Build‑Up の形     | ブレイク方向   | 推奨戦術          |
| ---------- | --------------- | -------- | ------------- |
| **上昇フラッグ** | 緩やかに下げる平行チャネル   | 上        | チャネル上限抜けでロング  |
| **下降フラッグ** | 緩やかに上げる平行チャネル   | 下        | チャネル下限抜けでショート |
| **ペナント**   | 収束する三角形（10〜20本） | 直前トレンド方向 | 三角形外へ逆指値      |

- **Measuring Move** : ブレイク後の目標値 = ポール（直前急伸/急落幅）の 50〜75 %。
- **EMA Interaction** : 25EMA がペナント中央を貫くほど理想的（圧力対称）。

## 3.5 スクイーズとエクスパンション — ボラティリティ・フェーズ

| フェーズ         | ATR挙動               | EMA 状態        | 戦略                          |
| ------------ | ------------------- | ------------- | --------------------------- |
| **スクイーズ**    | ATR20 が日中平均の 60 %以下 | EMA フラット & 内包 | ブレイク狙いの“待ち伏せ”のみ。逆張り厳禁。      |
| **エクスパンション** | ATR が 30 % 以上急伸     | EMA 傾き急角度     | トレンドフォロー優位。プルバック型セットアップへ切替。 |

> スクイーズ→エクスパンション移行直後は**勝率 60 %超**（書籍統計値）を示すゴールデンタイム。

## 3.6 キャンドル解析 — 実体とヒゲが語る圧力

### 圧力判定の 3 原則

1. **実体＞ヒゲ** → 主勢力の推進力が強い（Break / Follow‑Through 向き）
2. **ヒゲ＞実体** → 反対勢力の抵抗 or 利食いの兆候（Build‑Up 終盤）
3. **連続小実体** → エネルギー蓄積（ビルドアップ）

### シグナル足のチェックポイント

- “包み足（Engulfing）” なら方向確定感が高い
- “コンビ足（Inside + Break）” は迷い→加速の典型
- 長ヒゲ逆行足は **Double Pressure Release** → 即撤退シグナル

## 3.7 失速シグナルと手仕舞い判断

| シグナル                | 説明                     | アクション          |
| ------------------- | ---------------------- | -------------- |
| **フォロースルー<10 pips** | Break キャンドルの半分以下しか伸びない | 利益半分確保 or 全決済  |
| **連続上ヒゲ（下ヒゲ）**      | 反対勢力の成長を示唆             | ストップを BE へ繰上げ  |
| **EMA 再内包**         | 環境がレンジへ逆戻り             | ポジション解消 & 様子見へ |
